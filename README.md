# Reqon - Integrated Requisition Management System

**Final Year Project** | **April 2026**
Reqon: A lightweight web-based integrated multi-requisition management system for enterprise workflows.

## Tech Stack
- **Frontend**: HTML5, CSS3, Vanilla JavaScript (ES6+)
- **Backend**: PHP 7.4+
- **Database**: MySQL 5.7+
- **Server**: Apache (XAMPP)
- **Version Control**: Git/GitHub

## Scope
- Personnel Requisitions
- Procurement Requisitions  
- Maintenance Requisitions
- Merchandise Requisitions
